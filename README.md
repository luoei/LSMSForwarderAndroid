
## Android 短信转发，备用机小号专用，仅供学习使用。



```!
基于Java开发。

项目的目的是为方便个人有闲置Android机，小号等。

随着项目的使用情况和反馈，将时不时根据更新并完善用户体验与功能优化吗，欢迎提出问题。
```

### 工具简介
1. 优先使用WiFi发送
1. WIFI不可用使用短信转发
1. 黑名单标记后，还是会走发送，可在邮件收件处根据前缀【垃圾短信】过滤
1. 电信iFree卡短信签到，默认关闭。每隔20天会发送一条短信，为了免月租。
